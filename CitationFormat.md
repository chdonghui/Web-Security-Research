# 웹 페이지 출처 표기 방법

## 웹 페이지

- 저자명 (발행일자). 페이지명(기사제목). 웹사이트명. URL
- 발행일자 형식은 yyyy.mm.dd
- 웹 페이지에 따라 아래 중 선택
  - yyyy.mm.dd Published
  - yyyy.mm.dd Updated
  - yyyy.mm.dd Accessed
- URL은 마크다운 링크 형식으로
- 영문 저자명: 성, 이름 이니셜 가운데이름 이니셜
- 저자가 없는 경우: 기관명 (발행일자). 페이지명. URL
- 기관명과 웹사이트명이 다를 경우: 기관명 (발행일자). 페이지명. 웹사이트명. URL

## 위키피디아 페이지

- 제목 (발행일자). 위키백과. URL

# 책 출처표기 방법

- 저자명 (발행연도). 책제목. 출판사.
- 영문 저자명: 성, 이름 이니셜 가운데이름 이니셜
- 공동 저자: 저자명1, 저자명2 (발행연도). 책제목. 출판사.
- 특정 페이지 인용 시: 저자명 (발행연도). 책제목. 출판사. p.000 또는 pp.000-000

# 국제 표준 출처 표기 방법

## ISO/IEC 표준
- 기관명 (발행연도). 표준제목. 출판기관. 표준번호. URL

## RFC
- 저자명 (발행연도). RFC제목. IETF. RFC 0000. URL
- 저자가 여러 명인 경우: 저자명1, 저자명2 (발행연도). RFC제목. IETF. RFC 0000. URL
