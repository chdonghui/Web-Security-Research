# Security-Research

본 레파지토리는 보안에 필요한 기초 지식과 도구, 사용법을 정리합니다. 현재는 Web Security를 중점적으로 기술 중이며 향후 변경될 수 있습니다.

정리된 문서의 폴더 구조는 아래와 같습니다.

- Concept
- Tools
- Vulnerability

각 폴더의 기능은 아래와 같습니다.

**Concept:** 네트워크, 프로토콜, 웹 구조, 데이터베이스 등의 기초 개념을 기술합니다.

**Tools:** 보안 또는 개발에 필요한 도구들의 사용법과 기능을 살펴봅니다.

**Vulnerability:** 취약점 종류별로 개념과 예제, 공격 시나리오, 우회 방법, 대응 방안 등을 알아봅니다. 필요하다면 CVE를 추가해 구체적으로 실습해볼 예정입니다.
